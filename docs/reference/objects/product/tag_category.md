# Tag Category
{: .d-inline-block }
object
{: .label .fs-1 }

A tag category represents a grouping of tag values associated with a product. For example, a "Distance from venue" category might have values like "Walking distance" or "5 km".

#### Attributes

## `tag_category.category`
{: .d-inline-block }
string
{: .label .fs-1 }

The name of the tag category.

## `tag_category.values`
{: .d-inline-block }
array of strings
{: .label .fs-1 }

An array of the tag value names for this category that are associated with the product.
