# Redirects

The URL redirects are managed in **My Site > Redirects**, both Creators and Developers have access.

## URLs that can't be redirected

Redirects can't be set up **to** or **from** URL paths that start with:
- admin
- b
- blog
- bookings
- cart
- checkout
- common
- log-in
- log-out
- p
- products
- robots
- script
- shop
- shop-style
- sign-up
- style
- t
- terms
